# VTT jQuery Folder Plugin
A jQuery folder plugin.

## Usage

1. Include jQuery:

	```html
	<script src="path/to/jquery.min.js"></script>
	```

2. Include plugin's code:

	```html
	<script src="path/to/jquery.folder.js"></script>
	```

3. Call the plugin:

	```javascript
	$("#element").folder({
		// config
	});
	```
